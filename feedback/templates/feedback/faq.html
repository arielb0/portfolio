{% translate "Frecuently Asked Questions" %}

Te permite gestionar encuestas para saber que piensan las personas acerca de un tema en particular.

Acorde a tu imaginación y las características de tu negocio o investigación, el cielo es el límite. Puede ser empleada para toma de decisiones basadas en datos, investigación de mercado, inteligencia de negocio, saber que le gusta a tus clientes y actuar en base a ello.

Es gratis (no te costará dinero). Tampoco tiene anuncios (no te costará tu atención).

Si solo deseas responder una encuesta, no necesitas registrarte con un usuario. En caso de que desees gestionar encuestas, preguntas y respuestas, los principales componentes de esta aplicación, si es necesario pasar por el proceso de registro.

En la pagina principal, toca el botón llamado "Crear encuesta". Este te llevara a una pagina que contiene un formulario con los siguientes campos:

  • Titulo: El titulo con el cual se mostrara la encuesta (obligatorio).
  • Descripción: Permite agregar una breve descripción, que se mostrara en la sección de estadísticas de la encuesta (opcional).
  • Abierto: Este campo permite definir si los usuarios podrán los resultados de la encuesta (abierto) o estos serán visibles solo para el usuario que la creo (cerrado).

En el menú principal, da clic en el enlace nombrado "Preguntas". Este te llevara a una página que muestra todas las preguntas que has creado. Allí existe un botón llamado "Crear Pregunta". A continuación se mostrara un formulario con los siguientes campos:

  • Encuesta: Te permite seleccionar a que encuesta pertenece la pregunta (obligatorio).
  • Texto: Es el texto de la pregunta (obligatorio).
  • Indice: Mediante este campo seleccionas en que posición (orden) del formulario se mostrara la pregunta. De esta manera puedes controlar a voluntad el ordenen que deseas que las personas que responden el formulario vean las preguntas que responden.

  • Tipo: Este campo permite seleccionar que tipo de campo tendrá el formulario. Existen 4 opciones:

    • Selección: Útil cuando las personas pueden responder mediante opciones, pero solo pueden elegir una. Solo se muestra una opción a la vez.
    • Selección Radio: Similar a Selección, pero muestra todas las opciones.
    • Casilla de verificación: Permite seleccionar mas de una respuesta a la vez. Muestra todas las posibles opciones de respuesta.
    • Numérico: Este campo permite la entrada de datos numéricos.
  • Valor mínimo: Es el valor mínimo que podrá tener un campo numérico como entrada. Esta opción solo es valida para campos numéricos y sera ignorada para el resto.
  • Valor máximo: Es el valor máximo que podrá tener un campo numérico como entrada. Esta opción solo es valida para campos numéricos y sera ignorada para el resto.

En el menú principal, haz clic en el enlace nombrado "Respuestas". Este te llevara a una pagina que muestra un listado de respuestas que haz creado. Allí existe un botón llamado "Crear Respuesta". Al hacer clic en el, se mostrará un formulario que contiene los siguientes campos:

  • Pregunta: La pregunta a la cual estará vinculada la respuesta que estas creando.
  • Texto: El texto que se mostrara cuando el usuario seleccione esta opción como posible respuesta.

En la página de encuestas, para cada una, existe un enlace. Este te llevara a la pagina donde se muestra el formulario para responder la encuesta en cuestión.

En la página de encuestas, si la encuesta en cuestión es abierta, para todos los usuarios se mostrara un botón llamado "Estadísticas". Este te llevara a una página donde se muestran todos las estadísticas recopiladas de dos maneras principales: mediante gráficos y una tabla. Los gráficos te mostraran los resultados de una manera fácil e intuitiva, que permitirá tomar decisiones rápidamente. La tabla ayuda a entender mejor las respuestas obtenidas y a realizar análisis de forma mas profunda.

Por defecto, no se guardan datos sensibles que puedan revelar o ser enlazados a tu identidad (a menos que en la encuesta generada por terceros, los facilites a voluntad). Si tienes conocimientos de programación, puedes auditar el código y comprobarlo.

Habilidades

PostgreSQL

Sistema de gestion de base de datos, encargado de almacenar la informacion de las aplicaciones web. Simple, rapido, enfocado en cumplir los estandares SQL y de codigo abierto.

Java

Lenguaje de programacion orientado a objetos y plataforma informatica (OpenJDK). Posee tipado fuerte, que permite verificar el codigo antes que se ejecute, dentro del editor de codigo. Esto lo hace ideal para el desarrollo de aplicaciones web que ejecutan codigo en el lado del servidor (backend).

Spring Boot

Marco de desarrollo (framework) para programar aplicaciones en Java. Posee la implementacion mas completa para realizar una API REST o HATEOAS.

TypeScript

Lenguaje de programacion libre, superconjunto de Javascript. Emplea tipado fuerte para facilitar la deteccion y correccion de errores en grandes proyectos que involucren desarrollo de aplicaciones web.

HTML

Lenguaje de marcado para la web. Define la estructura de los principales elementos que conforman una aplicacion web.

CSS

Es un lenguaje que permite diseñar graficamente la interfaz de usuario de una aplicacion web.

Bootstrap

Marco de desarrollo para CSS. Permite añadir estilo o presentacion, sin tener que desarrollar todos los elementos graficos desde cero.

Angular

Marco de desarrollo para programar aplicaciones web del lado del cliente (frontend). Incrementa la interactividad y posibilita desarrollar aplicaciones de una sola pagina (SPA) y aplicaciones web progresivas (PWA).
